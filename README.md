My sister was jealous of my kindle, so i made her one… kinda\!

MyReader is a ebook reader with a built in book tracker\! It’s like a kindle without the hassle of having to use goodreads along side it.

You can upload books to read, epub or pdf, and read them right there and then. 

I love my kindle but it’s also lowkey expensive and using this is free\!

Try it out here: [https://k-p2606.github.io/my-reader/\#/](https://k-p2606.github.io/my-reader/#/)

Tech stack: TypeScript, React, Tailwind CSS, Dexie.js, IndexedDB, epub.js
